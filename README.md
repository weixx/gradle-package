# gradle-package
Android Studio 3.+ 使用gradle配置动态修改包名、图标、常量等，实现动态配置编译不同包名的APP和多渠道打包

##
环境Android Studio 3.1.3  + gradle 4.4 + build:gradle:3.1.3
